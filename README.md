# Lost and Found App 📱

## Project Description

The Lost and Found App is an Android application developed to help users report lost items and submit details about found items. This application makes it easier for people to recover their lost belongings by connecting users who lost items with users who found them.

This project is developed as a diploma-level academic microproject using Android Studio.

---

## Features 🚀

- Add lost item details
- Add found item details
- View list of lost items
- View list of found items
- Simple and user-friendly interface
- Easy navigation between screens

---

## Technologies Used 🛠️

- Java
- Android Studio
- XML (User Interface Design)
- SQLite / Firebase (Database)

---

## How to Run the Project ▶️

Follow these steps to run the application:

1. Download or clone the project from GitHub
2. Open Android Studio
3. Click on **Open Project**
4. Select the downloaded project folder
5. Connect your Android device or start Emulator
6. Click **Run**
